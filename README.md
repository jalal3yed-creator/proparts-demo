
# ProParts Investor Demo

This is a demo-only MVP for investors.

## Features
- VIN Search (Mock Data)
- Vehicle & Compatible Parts Display
- Clean UI
- No real database required

## Run locally
Backend:
  cd backend
  npm install
  node server.js

Frontend:
  cd frontend
  npm install
  npm run dev

## Deploy
- Backend → Render
- Frontend → Vercel
